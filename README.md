# Tik Tak Toe

Ein klassisches Tik-Tak-Toe Spiel mit lokalem Multiplayer, KI-Gegnern und Online-Multiplayer!

## 🎮 Spielmodi

### 👥 Lokaler 2-Spieler-Modus
Spiele mit einem Freund am selben Gerät.

### 🤖 KI-Modi
- **Einfach:** Spielt wie ein 5-Jähriger (macht Fehler)
- **Brutal:** Unschlagbar mit Minimax-Algorithmus

### 🌐 Online Multiplayer
Spiele gegen Freunde über das Internet mit Firebase!

## 🚀 Live Demo

Spiele direkt im Browser: [https://damagevindi.github.io/tik-tak-toe](https://damagevindi.github.io/tik-tak-toe)

## 🛠️ Lokale Installation

```bash
git clone https://github.com/Damagevindi/tik-tak-toe.git
cd tik-tak-toe
# Öffne index.html in einem Browser
```

## 🌐 Online Multiplayer Setup

Siehe [FIREBASE_SETUP.md](FIREBASE_SETUP.md) für detaillierte Anleitung.

Kurzversion:
1. `firebase-config.js.example` kopieren zu `firebase-config.js`
2. Deine Firebase-Werte eintragen
3. Fertig!

## ✨ Features

- 📱 **Responsive Design** - Funktioniert auf allen Geräten
- 🎨 **Modernes UI** - Gradienten, Animationen, Neumorphismus
- 🪙 **Münzwurf** - Zufällige X/O-Zuweisung in KI-Modi
- 📊 **Punktezähler** - Verfolge deine Siege
- 🔄 **Echtzeit-Sync** - Online-Züge ohne Verzögerung
- 🔒 **Sichere Config** - API-Keys nicht im Repository
