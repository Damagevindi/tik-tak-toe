# 🎮 Tik Tak Toe

Ein klassisches Tik-Tak-Toe Spiel für zwei Spieler - komplett in JavaScript, HTML und CSS.

## 🚀 Live Demo

Spiele direkt im Browser: [https://damagevindi.github.io/tik-tak-toe](https://damagevindi.github.io/tik-tak-toe)

## ✨ Features

- 🎯 **Zwei-Spieler-Modus** - Spieler X gegen Spieler O
- 📊 **Punktezähler** - Verfolge deine Siege und Unentschieden
- 🎨 **Modernes Design** - Schöne Gradienten und Animationen
- 📱 **Responsive** - Funktioniert auf Desktop und Mobilgeräten
- ⚡ **Schnell & Leicht** - Keine Installation nötig, läuft im Browser

## 🛠️ Technologien

- HTML5
- CSS3 (mit Gradienten, Animationen und Grid)
- Vanilla JavaScript (ES6+ Klassen)

## 🎮 Spielanleitung

1. Öffne die `index.html` in einem Browser
2. Spieler X beginnt immer
3. Klicke auf ein leeres Feld, um dein Symbol zu setzen
4. Wer als Erster 3 Symbole in einer Reihe hat, gewinnt
5. Klicke auf "Neues Spiel", um eine neue Runde zu starten

## 🏆 Gewinnkombinationen

```
0 | 1 | 2
---------
3 | 4 | 5
---------
6 | 7 | 8
```

- 3 Reihen (horizontal)
- 3 Spalten (vertikal)
- 2 Diagonalen
